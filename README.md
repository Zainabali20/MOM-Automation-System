# MOM Automation System

## Overview
Python-based Meeting Management System designed to automate creation and tracking of Minutes of Meeting (MOM).  
Reduces manual workload and improves reporting efficiency.

## Features
- User authentication
- Create and manage meetings
- Add attendees and discussion points
- Auto-generate summary report
- Export as PDF
- Email notifications
- Pagination & search

## Tech Stack
- Python 3.11
- Flask/Django (backend)
- MySQL (database)
- HTML/CSS/Bootstrap (frontend)
- GitHub for version control

## Installation
1. Clone the repo
2. `pip install -r requirements.txt`
3. Run `python main.py`

- Optimized database queries for faster performance
